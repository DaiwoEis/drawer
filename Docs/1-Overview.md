# 项目概述 (Project Overview)

## 1. 项目名称与简介
**项目名称**：Drawer (高性能绘图应用)

**简介**：
Drawer 是一个基于 Unity 引擎开发的高性能、低延迟的 2D 绘图应用程序。它旨在为用户提供流畅的数字绘图体验，支持压感笔触、高精度橡皮擦、多层撤销/重做以及跨分辨率的画布适配。项目采用领域驱动设计 (DDD) 思想，分离了核心业务逻辑与渲染表现层，确保了代码的可维护性与可扩展性。

**核心目标**：
- 提供接近原生的书写延迟与手感。
- 实现与分辨率无关的矢量级笔画存储。
- 构建稳定可靠的绘图状态管理（撤销/重做、序列化）。

## 2. 关键特性与亮点
- **高性能渲染**：
  - 基于 Command Buffer 的 GPU 绘图。
  - `ShaderVariantCollection` 预热机制，杜绝首笔卡顿。
  - 动态批处理与内存池优化 (Reusable Arrays)，实现零 GC 绘图循环。
- **高精度逻辑**：
  - 采用 `LogicPoint` (0-65535) 归一化坐标系，确保在不同尺寸屏幕上的一致性。
  - 基于空间哈希 (`StrokeSpatialIndex`) 的高效笔画碰撞检测（橡皮擦）。
  - 支持压力感应与笔锋平滑算法。
- **稳健架构**：
  - 分层架构 (App/Domain/Service/Presentation)。
  - 完整的命令模式 (Command Pattern) 实现撤销/重做。
  - 内置企业级可观测性基础设施 (`StructuredLogger`, `PerformanceMonitor`)。
- **分辨率自适应**：
  - 动态监听 Canvas 分辨率变化，自动调整逻辑世界比率 (`LogicToWorldRatio`)。

## 3. 目标用户与使用场景
- **目标用户**：
  - 数字艺术家、设计师。
  - 在线教育讲师（白板教学）。
  - 需要嵌入式绘图功能的工具开发者。
- **典型场景**：
  - 电子白板书写。
  - 图片标注与批改。
  - 简单的创意草图绘制。
